Модель зрошувальнї системи.

Ця модель основана на дискретному підході з певним тактом підрахунку значень рівнів води у каналах.
Виходячи з того, що кожен канал із тими, що виходять з него, повторює занальну структуру системи,
можна вважати, що система складаеться із підсистем подібних собі (фрактал).
Тобто для опису моделі для усієї системи достатньо описати модель для одного каналу.

Кожен канал має такі параметри:
climb - під'ем канала над попереднім (батьківським) каналом;
height - висота;
length - довжина;
width - ширина;
enginePower - потужність двигуна;
waterLevel - коефіцієнт рівеня води приймає значення [0; 1], множиться на height;
leak - коефіцієнт просочування води у землю (дно) за одиницю часу, множиться на waterLevel;
waterLevelLimits - проміжок нормального waterLevel, наприклад [0.4; 0.9]; якщо waterLevel не в цому поміжку,
	то треба або вмикати або вимикати насос відповідно до значення waterLevel;
children - список каналів, які качають воду з цього каналу (канали діти).

За кожен проміжок часу t виконується перерахування значень waterLevel всіх каналів з урахуванням всіх параметрів каналу.

Був проведений експеримент в якому початкові рівні води були нуловими в усіх каналах і проміжок часу t = 0.1 секунд.
Підраховано 100 значень waterLevel для кожного з каналів через кожний проміжок часу t.
Парвметри системи такі:

{
	"climb" : 1,
	"height" : 1.5,
	"length" : 8000,
	"width" : 10,
	"enginePower" : 100000,
	"leak" : 0.01,
	"waterLevelLimits" : {
		"lower" : 0.4,
		"higher" : 0.9
	},
	"waterLevel" : 0,
	
	"children":[
		{
			"climb" : 1,
			"height" : 1,
			"length" : 2530,
			"width" : 3,
			"enginePower" : 10000,
			"leak" : 0.01,
			"waterLevelLimits" : {
				"lower" : 0.2,
				"higher" : 0.8
			},
			"waterLevel" : 0,

			"children":[]
		},
		{
			"climb" : 1,
			"height" : 1.1,
			"length" : 1090,
			"width" : 2.7,
			"enginePower" : 1000,
			"leak" : 0.01,
			"waterLevelLimits" : {
				"lower" : 0.25,
				"higher" : 0.9
			},
			"waterLevel" : 0,

			"children":[]
		}
	]
}

Тобто це система де є головний канал і від нього відходять два канали.
